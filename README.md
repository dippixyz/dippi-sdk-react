# dippi-sdk-react

> Dippi sdk import components react

[![NPM](https://img.shields.io/npm/v/dippi-sdk-react.svg)](https://www.npmjs.com/package/dippi-sdk-react) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save dippi-sdk-react
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'dippi-sdk-react'
import 'dippi-sdk-react/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
